Projects
Simple RISC Machine
Microprocessor architecture that implements a Reduced Instruction Set Computer (RISC). Includes datapath, FSM controller, RAM, and memory-mapped I/O peripherals constructed using a DE1-S0C FPGA board.
Audio Mixer
Python scripts that process imported audio files and uses FFT and selective filtering to adjust sound.
Oven Reflow Controller
Embedded system using an EFM8 microcontroller to control a standard off-the-shelf 1500W toaster oven for reflow soldering.
Claw Feedback Control
System identification and feedback control implementation of a 3-finger claw using LTI modelling and PID control via MATLAB.
Baccarat Engine
Digital system that runs the game of Baccarat on a DE1-S0C FPGA board using seven-segment displays and red LEDS for user interface.
ARC4 Cracking Circuit
Digital system that cracks ARC4 encrypted messages by loading ciphertext on on-chip memory then initiates a "brute-force" attack by cycling through key space using FPGA implemented hardware.
Hilbert & Fourier Tranforms
Investigation and application of the Hilbert & Fourier transforms on DAQ signals for power analysis.
